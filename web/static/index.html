Guide

使用指引

按这个顺序操作，最容易把 OBS 画面和 Rebocap 评分一起跑起来。

1. 启动 OBS 虚拟摄像头

在 OBS 中点击“启动虚拟摄像头”，确认系统里已经暴露出 OBS Virtual Camera。

2. 允许浏览器访问摄像头

如果浏览器权限被拒绝，前端无法枚举本地视频设备，也无法自动识别 OBS。

3. 确认 Rebocap VMC 输出

默认监听 0.0.0.0:39539。设备监控页会显示 VMC FPS、UDP 包数量和骨骼数量。

4. 离线导入只对 NPY 自动评分

上传 NPY 会直接评估并写入历史。BVH/视频目前先归档，后续再扩展自动解析。
